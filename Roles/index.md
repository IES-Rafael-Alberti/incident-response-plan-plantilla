# Roles

A continuación, se presentan descripciones, deberes y entrenamiento para cada rol en una respuesta a incidentes.

## Estructura de Roles

*   Equipo de Mando
    *   [Incident Commander (IC)](rol-1-jefe-incidentes.md)
    *   [Incident Commander-Adjunto (Subjefe)](rol-2-subjefe-incidentes.md)
    *   [Escriba](rol-3-escriba.md)
*   [Equipo de Enlace](rol-5-enlace.md)
    *   Enlace Interno (Partes Interesadas de Secure Shield)
    *   Enlace Externo (Clientes Afectados, Reguladores, Proveedores como Microsoft Azure si es necesario)
*   [Equipo de Operaciones](rol-4-experto-materia-sme.md)
    * Expertos en la Materia (SME) para Plataforma Azure, Servidores Windows/Linux, Redes, Seguridad de Endpoints, Desarrollo Web (Portal Cliente).
    *   SME para Equipos/Unidades de Negocio (ej. Jefe de Pentesting si afecta operaciones).
    *   SME para Funciones Ejecutivas (Legal, Dirección, Comunicación).

Durante incidentes grandes y complejos, la estructura de roles puede ajustarse para crear subequipos.

Esta estructura es **flexible**: no todos los roles serán ocupados por una persona diferente en cada incidente. En un incidente pequeño, el Subjefe podría actuar como Escriba y Enlace Interno. La estructura se adapta según el incidente.

## Tiempo de Guerra vs. Tiempo de Paz

Durante las llamadas de respuesta a incidentes ("tiempo de guerra"), una estructura organizativa diferente anula las operaciones normales ("tiempo de paz"):

*   El Comandante de Incidentes (CI) está a cargo, con la máxima autoridad en la llamada, superando incluso al CEO.
*   Los respondedores primarios (actuando como principales en la llamada para un equipo/servicio) son las personas de mayor rango para ese servicio.
*   Las decisiones las toma el CI tras considerar la información. Una vez tomada, es final.
*   El CI puede tomar decisiones más arriesgadas que las normales en tiempo de paz.
*   El CI puede ir contra una decisión de consenso. Si 9/10 personas acuerdan algo y 1 disiente, el CI puede elegir la opción minoritaria. La decisión del CI es final.
*   El CI puede usar un lenguaje o comportamiento que parezca grosero. Es tiempo de guerra, y a veces la urgencia lo requiere. No es personal.
*   El CI puede pedirle que abandone la llamada o expulsarle. Está a discreción del CI si siente que no está proporcionando información útil. Recuerde que el tiempo de guerra es diferente.
